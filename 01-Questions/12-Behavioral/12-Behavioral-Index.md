# 12. Behavioral / Soft skills

STAR-истории, лидерство, конфликты, менторинг, code review, ownership.

## Типовые темы

- Конфликт с коллегой / тимлидом / PM — как разрешали
- Технический долг vs дедлайн — как приоритизировали
- Менторинг junior/middle, рост команды
- Code review: как даёте feedback, как принимаете
- Ошибка в продакшене — что было, как чинили, что сделали потом
- Принятие архитектурного решения (ADR) — процесс
- Лидерство без авторитета (cross-team)
- Работа с unclear requirements
- Найм / собеседования других
- Отказ от плохой идеи (своей или чужой)
- Burnout, work-life balance
- Почему уходите с текущего места? Почему сюда?

## Формат STAR

- **S**ituation — контекст
- **T**ask — задача
- **A**ction — что **ты** сделал
- **R**esult — измеримый результат + что вынес
