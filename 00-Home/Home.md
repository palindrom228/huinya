## ⚙️ Правила работы

1. **Каждое тестирование** = 12 вопросов, по одному на категорию (равномерное покрытие).
2. **Оценка вопроса 0–3**: 0 = не знал, 3 = ответил уверенно. Любая оценка < 2 → автоматически в [[04-Weak-Spots/README|Weak spots]] и заводится [[Template-SR-Card|SR-карточка]].
3. **Spaced repetition по SM-2**: интервалы 1 → 3 → 7 → 14 → 30 → 90 дней (с поправкой на ease).
4. **Mock-интервью** = раз в 1–2 недели, прогон системного дизайна + поведенческое.
5. **Язык:** объяснения по-русски, термины и код по-английски.

## 🚀 Quick start

1. Открой [[Dashboard]] — посмотри прогресс.
2. Создай новый тест: `Cmd+P → Templates: Insert template → Template-Test` (или вручную в `02-Tests/`).
3. Заполняй ответы, оценивай себя.
4. Слабые вопросы → [[04-Weak-Spots/README|Weak spots]] + новая [[Template-SR-Card|SR-карточка]].
